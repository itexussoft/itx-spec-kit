## Purpose

Use this extension command when phasing out existing capability in a
brownfield delivery slice. It is shipped by the local
`itx-brownfield-workflows` extension, not by upstream core commands.

## Expected planning artifact

Create or update:

`specs/<active-feature>/deprecate-plan.md`

Use the template at:

`.specify/templates/deprecate-plan-template.md`
